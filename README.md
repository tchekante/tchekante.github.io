# Резюме-сайт Антона Чеканова

Одностраничный сайт-резюме с переключателем RU/EN.

## Как открыть
Открой `index.html` в браузере. Переключатель языка — кнопки RU/EN справа вверху.
Ссылка на английскую версию: добавь `?lang=en` в конец URL.
Кнопка PDF печатает текущую версию (Ctrl/Cmd+P → Сохранить как PDF).

## Как разместить по ссылке

### Вариант 1 — Netlify Drop (быстро, 2 мин)
1. Зайди на app.netlify.com/drop
2. Перетащи папку `site` целиком в окно браузера
3. Получишь ссылку вида name.netlify.app

### Вариант 2 — GitHub Pages (солиднее)
1. Создай репозиторий на github.com
2. Загрузи `index.html` в корень
3. Settings → Pages → ветка main, папка /root → Save
4. Через минуту сайт по адресу username.github.io/repo-name

Файл self-contained: весь CSS и JS внутри index.html, внешних зависимостей нет.
